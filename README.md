# Tic Tac Toe (PyQt5)

Simple Tic Tac Toe game implemented with PyQt5.

Prerequisites
- Python 3.7+
- pip

Install dependencies (PowerShell):

```powershell
python -m pip install -r requirements.txt
```

Run the app:

```powershell
python app.py
```

Controls:
- Click an empty cell to place the current player's mark (X starts).
- Click "Reset" to clear the board.
